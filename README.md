# posn1exam

## Install

**For Unix/Linux**

```bash
npm install adm-zip fs-extra puppeteer
```

**For Windows 10/11**

```bash
cmd /c "npm install adm-zip fs-extra puppeteer"
```

## Setup

create `testhelper.user.json`

```json
{
  "username": ""
  "password": ""
}
```

## Usage

```bash
node testhelper.js ./.cph/.cpp_file_name.prob
```
